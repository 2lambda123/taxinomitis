Machine Learning for kids

What?
This tool introduces machine learning by providing hands-on experiences for training and building with machine learning systems.

It provides an easy-to-use guided environment for training machine learning models for classifying text, numbers or recognising images.

This builds on existing projects to introduce and teach coding to children, by adding these models to Scratch (a widely used educational coding platform), allowing children to create projects and build games with the machine learning models that they've trained.

The tool is entirely web-based and requires no installs or complicated setup to be able to use.

Why?
Machine learning is all around us. We all use machine learning systems every day - such as spam filters, recommendation engines, language translation services, chatbots and digital assistants, search engines, fraud detection systems, and much much more.

It will soon become normal for machine learning systems to drive our cars and be used by doctors in the diagnosis and treatment of major illnesses.

It's important that kids are aware of how the world around them works, and the best way to understand the capabilities and the implications is to be allowed to try building with this technology for themselves.
