Are you a teacher / group leader?

Machine Learning for Kids is designed for use in schools, however is also used by a variety of volunteer-led coding groups. If you create an account for your school or coding club, you can create or manage logins for individual students yourself, within that class account.
If you are a teacher or coding group leader who would like access for your group, create a class account below.
There are two types of class account depending on who sets it up.
Create a managed class account
If you would like Dale to set up the class account, use the button below to send an email with the following information:
  • When you would like the account for
  • Roughly how many student accounts you want to be able to have
  • The name of the school or coding group the account is for
Create an unmanaged class account
If you are happy to set up and manage the class account yourself, you can do it now!
This will involve you:
  • Creating an account on IBM Bluemix
  • Creating API key(s) for Watson cloud services needed to train machine learning models
  • Entering the API keys into this Machine Learning for Kids tool
Step-by-step instructions with screenshots are available to walk you through this.
