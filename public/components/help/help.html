Known problems

These are bugs that I know about and am working on fixes for. Sorry! If you spot any bugs I don't know about, please let me know.

'text' projects

Not able to store some accented characters in examples in text projects
Most accented characters work fine, but there are a small number of characters like ğ which don't store. (You'll get "Error accessing the database used to store data" every time you try). I am working on a fix for this, but in the meantime, please just use the un-accented equivalents for any letters that give you problems. Sorry!

'images' projects

Dragging images into the training buckets doesn't work if you're using Safari or Internet Explorer
Sorry about this. If you can use another browser like Firefox or Chrome, please do that. Otherwise, you can click the "Add example" button and paste in the URL of training pictures instead.
