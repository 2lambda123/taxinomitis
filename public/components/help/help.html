Dragging images into the training buckets doesn't work if you're using Safari or Internet Explorer
Sorry about this. If you can use another browser like Firefox or Chrome, please do that. Otherwise, you can click the "Add example" button and paste in the URL of training pictures instead.
Errors about 'unsupported file type' when training a model
You can only use jpg or png images to train the computer with. I haven't got the web tool to help you make sure you only collect jpg and png pictures yet, so you need to try and be careful for now. I will fix this though so the training tool makes sure you collect the right files only.
